suivant un procédé géométrique.

Il est possible de traduire le procédé de Dom Bedos afin d'obtenir des données purement numériques. Ceci est particulièrement utile si l'on souhaite automatiser les calculs au moyen d'un tableur ou de n'importe quel autre programme informatique. On passera alors par une fonction dont le procédé de calcul est entièrement basé sur la propriété homothétiques des triangles rectangles.




B = (
L2
(
L1
(A - C)
)
) + C

Où les valeurs sont représentées par :

  • A : le premier diamètre.
  • B : le diamètre recherché.
  • C : le dernier diamètre.
  • L1 : une distance quelquonque figurée entre A et C.
  • L2 : l'écartement entre les deux diamètre fixé par la règle de division.
L'avantage de cette formule réside dans le fait que B peut aussi se trouver tant à gauche de A qu'à droite de C (sur le graphique).
