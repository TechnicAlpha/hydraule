Pour diviser la troisieme octave, prenez le point du milieu entre X & 14 ; ce point sera C sol ut diese, 26.
Prenez le milieu entre X & 15 ; ce point sera D la re, 27.
Prenez le milieu entre X & 16 ; ce point sera E si mi bemol, 28.
Prenez le milieu entre X & 17 ; ce point sera E si mi, 29.
Prenez le milieu entre X & 18 ; ce point sera F ut fa, 30.
Prenez le milieu entre X & 19 ; ce point sera F ut fa diese, 31
Prenez le milieu entre X & 20 ; ce point sera G re sol, 32.
Prenez le milieu entre X & 21 ; ce point sera G re sol diese, 33.
Prenez le milieu entre X & 22 ; ce point sera A mi la, 34.
Prenez le milieu entre X & 23 ; ce point sera B fa si bemol, 35.
Prenez le milieu entre X & 24 ; ce point sera B fa si, 36.
Prenez le milieu entre X & 25 ; ce point sera C sol ut, 37.

On fera de même pour la quatrieme Octave.

Prenez le milieu entre X & 26 ; ce point sera C sol ut diese, 38.
Prenez le milieu entre X & 27 ; ce point sera D la re, 39.
Prenez le milieu entre X & 28 ; ce point sera E si mi bemol, 40.
Prenez le milieu entre X & 29 ; ce point sera E si mi, 41.
Prenez le milieu entre X & 30 ; ce point sera F ut fa, 42.
Prenez le milieu entre X & 31 ; ce point sera F ut fa diese, 43.
Prenez le milieu entre X & 32 ; ce point sera G re sol, 44.
Prenez le milieu entre X & 33 ; ce point sera G re sol diese, 45.
Prenez le milieu entre X & 34 ; ce point sera A mi la, 46.
Prenez le milieu entre X & 35 ; ce point sera B fa si bemol, 47
Prenez le milieu entre X & 36 ; ce point sera B fa si, 48.
Prenez le milieu entre X & 37 ; ce point sera C sol ut, 49.
Prenez le milieu entre X & 38 ; ce point sera C sol ut diese, 50
Prenez le milieu entre X & 39 ; ce point sera D la re, 51.
